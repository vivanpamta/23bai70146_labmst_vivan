# React Counter Application

**Experiment:** Question 1 – Counter Application using useState


 **Aim**
To create a simple React Counter Application that demonstrates:
- **Dynamic UI updates** using React state  
- **Event handling** with buttons  
- **Functional components** structure in React  

---

## **Description**
This project is a basic counter app built with React.  It displays a number on the screen and allows the user to:
- Increment the value  
- Decrement the value  
- Reset the value to zero  

The counter value is managed using the **useState hook**, which ensures the UI updates automatically whenever the state changes.

---

## **Features**
- Increment, Decrement, and Reset buttons  
- Styled UI with modern buttons and gradient background  
- Component-based architecture for reusability  

---
